7. `Glob **/CHANGELOG.md` → 1 in repo root, 1 vendored under `ignore/`.
8. `go doc -all <pkg>` for every one of the 180 packages → tabulated symbol counts.
9. `Grep -E 'T1[0-9]+(\.[0-9]+)?' --include='*.go' --include='*.md'` → reconciled with `docs/mitre.md`.
10. Read of `README.md`, `doc.go`, `docs/mitre.md`, `docs/architecture.md`, `docs/getting-started.md`, plus 8 representative `docs/techniques/*/*.md` pages.

## 7. Recommendations — target shape

### 7.1 Root README (proposed)

Length budget: ~250 lines max.

1. `# maldev` + 1-line tagline + badges.
2. `## Why?` (≤ 10 lines) — operator + researcher + student pitch.
3. `## Install` (≤ 5 lines).
4. `## Quick Start` (≤ 30 lines).
5. `## Where to start` (≤ 30 lines) — **3 explicit reading paths**: (a) "I want to write an implant" → `docs/getting-started.md`. (b) "I want to understand a technique" → `docs/techniques/<area>/<technique>.md`. (c) "I want the API for one package" → `pkg.go.dev/...`.
6. `## Package map` (≤ 30 lines) — **2-column tree** category → 5 packages max with one-liner. Move the dense pipe-delimited list to `docs/index.md`.
7. `## Build` (≤ 10 lines).
8. `## Acknowledgments`, `## License`.

Critical fix: the 5 broken links at line 85.

// Package <name> <one-sentence purpose>.

### 7.3 Per-technique markdown template (proposed)

1. `# <Title>` (H1 only).
2. `## TL;DR` — 3 lines.
3. `## Primer` — 100-200 words.
4. `## How It Works` — diagrams + steps.
5. `## API Reference` — godoc link + brief signatures.
6. `## Examples`:
   - `### Simple` — minimum-viable runnable.
   - `### Composed` — combined with ≥1 other package.
   - `### Advanced` (optional) — full chain.
7. `## OPSEC & Detection` — artefacts left, defender view, D3FEND tag.
8. `## MITRE ATT&CK` — table `T-ID | Name | Sub-coverage | D3FEND counter`.
9. `## Limitations` — Windows version, admin/SYSTEM, AV detections.
10. `## Compared to Other Implementations` — Cobalt Strike, Mythic, Nim.
11. `## See also` — siblings, doc.go anchor, references.

Length budget: 200–400 lines per page.

Pick **one** name: `### Composed` (eliminate "Combined Example" / "Real beacon loop").

### 7.4 Index / navigation spine (proposed)

Create **`docs/index.md`** with 4 entry-points:

1. **By role** → "Operator", "Red team researcher", "Detection engineer / blue", "Student".

3. **By MITRE ATT&CK ID** → reverse-index (auto-generatable from grep).

```md

```
